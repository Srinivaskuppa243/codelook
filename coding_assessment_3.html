Question 3

Write a function isPrime(n) hat checks if a given number n is a prime number. A prime number is a number greater than 1 that has no divisors other than 1 and itself.

Constraints:
  1. The input number n will be between 1 and 10^6.
  2. You need to check if the number is prime with an efficient approach.

Sample Testcase: 
                isPrime(5):
                Output:True
                Explantion: 5 is a prime number.
                
                isPrime(4):
                Output: false
                Explanation: 4 is not a prime number 
                because it is divisible by 2.
