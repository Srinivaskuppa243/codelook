The following code is intended to calculate the factorial of a number n, but it contains errors. Debug the code to make it work as expected. 

                function factorial(n) {
                    if (n = 0) { 
                        return 1;
                    }
                    return n * factorial(n - 1); 
                }
                
                console.log(factorial(5));
Constraints:
  1. n will be a non-negative integer
  2. n will be within the range 0≤n≤20.
  3. Ensure the input is always an integer. If the input is not an integer, the function is not required to handle it.
