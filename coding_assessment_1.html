Question 1

Write a function to reverse a string. Follow the constraints:

Constraints:
  1. No Use of Built-in Reverse Methods
  2. The time complexity should be O(n), where n is the length of the string.
  3. The input string is guaranteed to be non-null and its length will be between 1 and 10^5.

Sample Testcase: 
                Input: "hello"
                Output: "olleh"
                
                Input:"world"
                Output:"dlrow"
